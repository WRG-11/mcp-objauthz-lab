# Challenge S4 — The Magic Word

**Difficulty:** Beginner  
**Class:** CWE-639 — Wildcard/sentinel value bypasses scope filter  
**Tool under test:** `note_export`

## Setup

```bash
npm install
LAB_S4=vuln node src/server.js
```

## Your mission

You are **Alice** (`alice-token`, org *Acme*). The lab has three organizations
— Acme, Globex, Initech — each with two private notes (six total).

`note_export` exports an organization's notes. When called without arguments,
it exports your own org's notes (2 notes for Alice).

Your goal: with a single `note_export` call, retrieve all **six** notes from
all three organizations simultaneously.

There is a parameter that a comment in the tool description hints is for
"admin use." You are not an admin. Try it anyway.

## Constraints

- Token: `alice-token` only.
- `LAB_S4=vuln` must remain set.
- One tool call only.

## Verification

The exploit succeeds when `note_export` returns exactly **6 notes** covering
all three orgs (`org_acme`, `org_globex`, `org_initech`).
