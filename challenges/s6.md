# Challenge S6 — The Uninvited Guest

**Difficulty:** Intermediate
**Class:** CWE-639 — Foreign-parent injection
**Tool under test:** `note_create_in_org`

## Setup

```bash
npm install
LAB_S6=vuln node src/server.js
```

## Your mission

You are **Alice** (`alice-token`, org *Acme*). The lab exposes a
cross-team collaboration tool, `note_create_in_org`, that lets a caller
create a note "inside" a specified organization.

You are not a member of *Globex* (`org_globex`) — you have no Globex
token, no Globex membership, nothing.

Your goal: get a note of your choosing to show up in **Globex's**
`note_list` results, using only your own `alice-token`.

## Constraints

- Token: `alice-token` only.
- `LAB_S6=vuln` must remain set.
- One `note_create_in_org` call is enough.

## Verification

The exploit succeeds when the note returned by `note_create_in_org` has
`"orgId": "org_globex"`, even though the call was authenticated as Alice
(org Acme).
