# Challenge S2 — The Obedient Parameter

**Difficulty:** Beginner–Intermediate  
**Class:** CWE-639 — Client-supplied scope trusted as authorization  
**Tool under test:** `note_search`

## Setup

```bash
npm install
LAB_S2=vuln node src/server.js
```

## Your mission

You are **Alice** (`alice-token`, org *Acme*). You want to read notes that
belong to **Globex** (`org_globex`) — an organization you have no membership
in.

You will use only `note_search`. No other tool is needed.

1. Call `note_search` normally (no special arguments). Observe what you get.
2. Find the argument that makes the server search a *different* org's notes.
3. Read Globex's notes using only your own token.

## Constraints

- Token: `alice-token` only.
- `LAB_S2=vuln` must remain set.
- Do not call any other tool.

## Verification

The exploit succeeds when `note_search` returns a note with `"orgId":
"org_globex"` while you are authenticated as Alice.
